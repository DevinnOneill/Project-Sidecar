# Session Logs

> Append-only session log repository. Each file follows the format defined in `Gemini.md` Section 9.

## Naming Convention

```
YYYY-MM-DD_SC-YYYY-MMDD-NNN.md
```

## Rules

- One file per session
- Append-only — never edit or delete a session log
- Every session (clean completion or halt) produces a log entry
- Log format is defined in `Gemini.md` Section 9
